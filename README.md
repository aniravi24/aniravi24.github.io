# Personal website

Built with [Astro](https://astro.build/), deployed to GitHub Pages at [aniravi.com](https://aniravi.com).

## Develop

```sh
pnpm install
pnpm dev      # local dev server
pnpm build    # static build to dist/ (+ Pagefind search index)
```

Posts live in `src/content/blog/*.md`. Pushing to `main` builds and deploys automatically.
